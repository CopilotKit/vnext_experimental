## Overview

The Agentic Chat UI provides a complete chat interface that connects to your AI agents. It handles messages, tool execution, and real-time updates automatically.

## Quick Start

<Tabs>
  <Tab title="React">
```tsx

  return (
    <CopilotKitProvider runtimeUrl="/api/copilotkit">

      </div>

```
  </Tab>

## Custom Input

<Tabs>
  <Tab title="React">
```tsx
function CustomInput({ onSubmit, isDisabled }) {
  const [value, setValue] = useState("");

  return (
    <div>
      <input
        value={value}
        onChange={(e) => setValue(e.target.value)}
        disabled={isDisabled}
      />
      <button
        onClick={() => {
          onSubmit(value);
          setValue("");
        }}
      >
        Send
      </button>
    </div>
  );
}

<CopilotChat inputComponent={CustomInput} />
```
  </Tab>

  <Tab title="Angular">
```typescript
@Component({
  selector: "app-custom-input",
  template: `
    <div>
      <input [(ngModel)]="value" [disabled]="isDisabled()" />
      <button (click)="send()">Send</button>
    </div>
  `,
})
class CustomInputComponent {
  value = "";
  isDisabled = input<boolean>();
  chatState = inject(ChatState);

  async send() {
    await this.chatState.submitMessage(this.value);
    this.value = "";
  }
}
```

```html
<copilot-chat [inputComponent]="CustomInputComponent"></copilot-chat>
```
  </Tab>
</Tabs>
