## Getting Started

This quickstart guide will help you set up and run CopilotKit vnext_experimental in your development environment.

### Prerequisites

Make sure you have the following installed:

- **Node.js** (version 18 or higher)
- **pnpm** (recommended package manager)

<CodeGroup>

```bash npm
npm install -g pnpm
```

```bash yarn
yarn global add pnpm
```

```bash brew
brew install pnpm
```

</CodeGroup>

## Installation

<Steps>
  <Step title="Clone the Repository">
    Clone the CopilotKit vnext_experimental repository to your local machine:

    ```bash
    git clone https://github.com/copilotkit/copilotkit2.git
    cd copilotkit2
    ```

  </Step>
  
  <Step title="Install Dependencies">
    Install all dependencies using pnpm:

    ```bash
    pnpm install
    ```

  </Step>
  
  <Step title="Build the Project">
    Build all packages in the workspace:

    ```bash
    pnpm build
    ```

  </Step>
  
  <Step title="Run Tests">
    Verify everything is working by running the test suite:

    ```bash
    pnpm test
    ```

  </Step>
</Steps>

## Development Commands

Here are the key commands you'll use during development:

<CodeGroup>

```bash Development
# Start development mode
pnpm dev

# Build all packages
pnpm build

# Run tests
pnpm test

# Watch mode for tests
pnpm test:watch
```

```bash Linting & Formatting
# Run linting
pnpm lint

# Format code
pnpm format

# Type checking
pnpm check-types
```

</CodeGroup>

## Project Structure

```
CopilotKit2/
├── packages/
│   ├── runtime/          # Server-side runtime
│   ├── shared/           # Common utilities
│   ├── eslint-config/    # Shared ESLint configs
│   └── typescript-config/ # TypeScript presets
├── apps/
│   └── docs/            # Documentation (Mintlify)
└── turbo.json           # Turborepo configuration
```

## Using the Runtime Package

The runtime package provides server-side utilities for handling AI interactions:

```typescript
import { createRuntime } from "@copilotkitnext/runtime";

const runtime = createRuntime({
  // Configuration options
});

// Handle AI requests
runtime.handle(request);
```
